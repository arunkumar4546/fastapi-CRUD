# 🚀 FastAPI Basic CRUD Project (Without Database)

This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using **FastAPI**.  
The data is stored in an in-memory Python list instead of a database.

> ⚠️ Note: Data will be lost when the server restarts.

---

## 📌 Features

- Create a new record (POST)
- Read all records (GET)
- Read a record by roll number (GET with path parameter)
- Update a record (PUT)
- Delete a record (DELETE)
- Proper error handling using HTTPException

---

## 🛠 Technologies Used

- Python 3.x
- FastAPI
- Pydantic
- Uvicorn
